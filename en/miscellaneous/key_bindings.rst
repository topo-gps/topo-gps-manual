Keyboard shortcuts
==================
On MacOS and iPadOS it is possible to control the app using the keyboard. Below the most important keyboard commands are listed for MacOS and iPadOS.

Mac OS
------
- Escape: Cancel, or close pop-up window

- -/-: Zoom the map out
- +/=: Zoom the map in
- Option-↑: Zoom the map out
- Option-↓: Zoom the map in
- Arrow keys: Scroll the map

- Command-drag: Hold the command button together with the mouse/trackpad button to select an map area
- Shift-drag: Hold the shift button together with the mouse/trackpad button and move upwards to zoom in, and outwards to zoom out.

- Command-A: Creates a point at the location of the pointer
- Command-B: Enter coordinates
- Command-C: Center map
- Command-D: Create compass direction
- Command-E: Export route, point or layer
- Command-F: Search a location
- Command-G: Generate route
- Command-H: Hide window
- Command-I: Import file
- Command-J: Switch to planning screen
- Command-K: Switch to main map screen
- Command-L: Show route, point, or layer on map
- Command-Shift-L: Remove route, point or layer from map
- Command-M: Minimize window
- Command-N: New copied window. The contents of the current map screen is copied to the new window.
- Command-Shift-T: New empty window. The new window is opened without any geodata loaded.
- Command-P: Print. It will open the print pop-up for the currently visible area
- Command-R: Shows map ruler, for measuring distances, angles and height profiles.
- Command-Q: Quit. It will quit Topo GPS
- Command-S: Save. Saves a modification
- Command-Shift-S: Opens or closes the sidebar
- Command-T: New copied tab. The contents of the current map screen is copied to the new tab.
- Command-Shift-T: New empty tab. The new tab is opened without any geodata loaded
- Command-W: Close window or tab
- Command-Z: Undo
- Command-,: Settings

- Command-1: Opens saved routes screen
- Command-2: Opens shared routes screen
- Command-3: Opens waypoints screen
- Command-4: Opens layers screen

- Control-1: Switches the map to the main topographic map
- Control-2: Switches the map to an alternative topographic map of same region, if available
- Control-3: Switches the map to OpenStreetMap
- Control-4: Switches to aerial imagery, if available.
