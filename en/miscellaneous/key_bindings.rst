Key bindings
============
On MacOS and iPadOS it is possible to control the app using the keyboard. Below the most important keyboard commands are listed.

Mac OS
------
- Command-A: Creates a point at the location of the pointer
- Command-B: Enter coordinates
- Command-C: Center map
- Command-D: Create compass direction
- Command-E: Export route, point or layer
- Command-F: Search a location
- Command-G: Generate route
- Command-H: Hide window
- Command-I: Import file
- Command-J: Switch to planning screen
- Command-K: Switch to main map screen
- Command-L: Show route, point, or layer on map
- Command-Shift-L: Remove route, point or layer from map
- Command-M: Minimize window
- Command-N: New window
- Command-P: Print. It will open the print pop-up for the currently visible area
- Command-Q: Quit. It will quit Topo GPS
- Command-S: Save. Saves a modification
- Command-T: New tab
- Command-W: Close window or tab
- Command-Z: Undo
- Command-,: Settings
</APPLE>
