Units
=====
In the section units of the :ref:`settings <ss-settings>` you can set the length/speed, angle and slope units.

The length/speed can be set to Metric (m, km, km/h), Imperial (ft, mi, mph) or Nautical (ft, NM, kt).

The unit of angle can be set to Degrees (°, 360), Gon (400), NATO mil (6400), Miliradians (6283), Swedish streck (6300) and Finish streck (6000). The number between parentheses indicates the amount of units fit in a full circle.

The slope unit can be set to percentage (%) or degrees (°).
